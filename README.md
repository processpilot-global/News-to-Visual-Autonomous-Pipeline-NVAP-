# News-to-Visual-Autonomous-Pipeline-NVAP-
NVAP v1.0: News-to-Visual Autonomous Pipeline by ProcessPilot.ai. An n8n engine that ingests RSS feeds, distills news via GPT-4o mini, and synthesizes 9:16 cinematic thumbnails via DALL-E 3. Features automated SMTP email alerts upon asset completion. Built by Abdullah Butt to bridge the gap between real-time data and viral visual assets.
